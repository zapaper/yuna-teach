# Sounds

Drop audio files here for in-app sound effects. Files referenced by the app:

- `cheer.mp3` — plays when confetti fires on the review page (student scores ≥ 90%). A soft crowd-cheer, kept short (2-3s) and mixed quietly. Any royalty-free cheer works; look for sources like mixkit.co or pixabay.com.

If a file is missing the code silently no-ops — no errors, no crash.
